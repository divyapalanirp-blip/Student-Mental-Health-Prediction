# Student Mental Health Prediction using Machine Learning

## Objective
To build a machine learning model that predicts whether a student has depression based on mental health and academic factors.

## Dataset
Student Mental Health Dataset

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Google Colab

## Machine Learning Algorithm
Decision Tree Classifier

## Steps Performed
1. Loaded the cleaned dataset
2. Converted categorical data into numerical values
3. Split data into training and testing sets
4. Trained a Decision Tree model
5. Made predictions on test data
6. Calculated accuracy
7. Generated a confusion matrix

## Results
The model was successfully trained and evaluated using accuracy score and confusion matrix.

## Conclusion
This project demonstrates the use of supervised machine learning for predicting student mental health outcomes.
